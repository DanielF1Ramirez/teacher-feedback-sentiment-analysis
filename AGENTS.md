# AGENTS.md

## Scope
This repository is the only editable project in the current workspace.

## Project
Professional NLP repository for teacher feedback sentiment analysis, based on the original `PLN` source material.

## Working rules
1. Keep changes small, safe, and reviewable.
2. Do not refactor unrelated modules.
3. Do not edit notebooks unless explicitly requested.
4. Keep comments and docstrings in English.
5. Keep commit messages in English.
6. Prefer modular Python code under `src/`.
7. Keep entrypoint scripts under `scripts/`.
8. Update documentation when behavior changes.
9. Avoid unnecessary dependencies.
10. Prefer fast validations and tests.
11. Do not run heavy training unless explicitly requested.
12. Before touching multiple files, briefly explain the intended scope.

## Repository goals
- Clean repository structure
- Reproducible environment
- Professional documentation
- Clear preprocessing, feature engineering, training, evaluation, and inference flow
- Portfolio-ready presentation

## Definition of done
- README is professional and accurate
- Dependencies are aligned
- Main scripts are coherent
- Lightweight tests exist for important logic
- CI scaffold is present
- No obvious placeholders remain
- Repo is ready for GitHub publication
