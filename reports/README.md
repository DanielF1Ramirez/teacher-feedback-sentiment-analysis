# Reports Directory

## Purpose

Stores generated evaluation artifacts and experiment summaries.

## Current Contents

- `metrics/baseline_metrics.json`: baseline run metrics generated by `scripts/run_baseline.py`.
- `metrics/model_selection_metrics.json`: generated by `scripts/run_experiment.py` when the classical ML comparison workflow is executed locally.

## Reproducibility

Regenerate baseline metrics with:

`python scripts/run_baseline.py`

Regenerate the full model-selection report with:

`python scripts/run_experiment.py`
